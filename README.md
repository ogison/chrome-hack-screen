# Hacking Animation Screen - Chrome拡張機能

画面にハッキング風のクールなアニメーションを表示するChrome拡張機能です。

## 概要

このChrome拡張機能は、ツールバーのアイコンをクリックするだけで、現在表示中のWebページ上にハッキングされたような視覚効果のアニメーションを表示します。プレゼンテーション、デモンストレーション、エンターテイメント目的で使用できます。

## 主な機能

- **マトリックス風アニメーション**: 緑色の文字が画面を流れ落ちる
- **グリッチエフェクト**: 画面が歪む視覚効果
- **ターミナル風インターフェース**: ハッキングコマンドが実行されているような演出
- **カスタマイズ可能**: 複数のスタイル、カラースキーム、継続時間を設定可能
- **簡単操作**: ESCキーまたはクリックで即座に終了

## インストール方法

（開発中）

1. このリポジトリをクローン
2. Chrome拡張機能ページ（`chrome://extensions/`）を開く
3. デベロッパーモードを有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. プロジェクトフォルダを選択

## 使い方

1. Chrome拡張機能のツールバーアイコンをクリック
2. ハッキングアニメーションが自動で開始
3. ESCキーまたは画面をクリックして終了
