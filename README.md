# Hacking Animation Screen - Chrome拡張機能

[![CI](https://github.com/ogison/chrome-hack-screen/actions/workflows/ci.yml/badge.svg)](https://github.com/ogison/chrome-hack-screen/actions/workflows/ci.yml)
[![Release](https://github.com/ogison/chrome-hack-screen/actions/workflows/release.yml/badge.svg)](https://github.com/ogison/chrome-hack-screen/actions/workflows/release.yml)

画面にハッキング風のクールなアニメーションを表示するChrome拡張機能です。

## 概要

このChrome拡張機能は、ツールバーのアイコンをクリックするだけで、現在表示中のWebページ上にハッキングされたような視覚効果のアニメーションを表示します。プレゼンテーション、デモンストレーション、エンターテイメント目的で使用できます。

## 主な機能

### アニメーション効果

- **マトリックス風アニメーション**: 緑色の文字（カタカナ、数字、記号）が画面を流れ落ちる
- **グリッチエフェクト**: RGB色ずれ、スキャンライン、画面の歪みなどの視覚効果
- **ターミナル風インターフェース**: ハッキングコマンドがタイピングされるアニメーション

### 操作性

- **ワンクリック起動**: ツールバーアイコンをクリックするだけで即座に開始
- **簡単終了**: `ESC` キーまたは画面クリックで即座に終了
- **カスタマイズ可能**: オプション画面で継続時間、スタイル、カラースキームを設定

### セキュリティ

- **プライバシー保護**: ユーザーデータの収集なし
- **安全性**: ページコンテンツの改変なし（視覚的なオーバーレイのみ）
- **クリーンアップ**: アニメーション終了後、すべての要素を完全に削除

## インストール方法

### 開発版のインストール

1. このリポジトリをクローン
```bash
git clone https://github.com/ogison/chrome-hack-screen.git
cd chrome-hack-screen
```

2. Chrome拡張機能ページを開く
   - アドレスバーに `chrome://extensions/` と入力

3. デベロッパーモードを有効化
   - ページ右上の「デベロッパーモード」トグルをONにする

4. 拡張機能を読み込む
   - 「パッケージ化されていない拡張機能を読み込む」をクリック
   - プロジェクトのルートフォルダを選択

5. インストール完了
   - ツールバーに拡張機能のアイコンが表示されます

## 使い方

### 基本的な使い方

1. **アニメーション開始**
   - Chrome拡張機能のツールバーアイコンをクリック
   - ハッキングアニメーションが自動で開始されます

2. **アニメーション終了**
   - `ESC` キーを押す、または
   - 画面の任意の場所をクリック

### 設定のカスタマイズ

1. 拡張機能アイコンを右クリック
2. 「オプション」を選択
3. 以下の設定が可能です：
   - **アニメーション継続時間**: 5〜60秒
   - **アニメーションスタイル**: ミックス、マトリックス風、ターミナル風、グリッチ風
   - **カラースキーム**: 緑、赤、青（開発予定：カスタム）
   - **サウンドエフェクト**: ON/OFF（開発予定）

## 制限事項

- 保護されたページ（`chrome://`、`edge://` など）では動作しません
- 非常に重いページでは若干のパフォーマンス低下の可能性があります
- iframeの内部には適用されません

## トラブルシューティング

### アニメーションが表示されない

- 保護されたページでないか確認してください
- 拡張機能が有効になっているか確認してください（`chrome://extensions/`）
- ブラウザのコンソール（F12）でエラーメッセージを確認してください

### アニメーションが終了しない

- `ESC` キーを押すか、画面をクリックしてください
- ブラウザのタブを閉じることでも終了できます

### 設定が保存されない

- ブラウザのストレージ権限が有効になっているか確認してください
- 拡張機能を再インストールしてみてください
